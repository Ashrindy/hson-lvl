# `hson-lvl`
***C++ editor for HSON (set editor)***
# Description
A universal HSON (Hedgehog Set Object Notation) editor made in C++ powered by [HedgeLib](https://www.github.com/Radfordhound/HedgeLib), [imgui](https://www.github.com/ocornut/imgui), [SDL3](https://www.libsdl.org/) and [plume](https://www.github.com/renderbag/plume) (Vulkan, D3D12).
# WARNING
This editor is still under rapid development, most features aren't finalized neither polished.
